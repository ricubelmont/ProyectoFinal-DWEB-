Tipos de Encabezados en HTML

Encabezado H1

Este es el encabezado de nivel 1. Se utiliza generalmente para el título principal de la página.

Encabezado H2

Este es el encabezado de nivel 2. Se usa como subtítulo o para dividir secciones principales.

Encabezado H3

Encabezado de tercer nivel, ideal para subsecciones dentro de un H2.

Encabezado H4

Un nivel más profundo, útil para estructurar aún más tu contenido.

Encabezado H5

Se usa con poca frecuencia, pero puede ayudar a organizar contenido complejo.

Encabezado H6

El nivel más bajo de encabezado, con menor peso visual.
